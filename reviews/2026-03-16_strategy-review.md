# 戦略多視点レビュー（2026-03-16）

運用期間: 2026-03-04 〜 03-16（約2週間）
成績: 9戦1勝8敗（勝率12.5%）、合計 -¥7,577（-2.5%）

---

## 1. テクニカルアナリスト

### 良い点
- トレーリングストップ(-5%) + 段階的利確(+7%)はリスク管理の基本として正しい
- SMA200の長期トレンドフィルターは理論的に正しいアプローチ
- 安川電機で+7%利確ルールが正常に発動

### 問題点
- **ゴールデンクロスの遅延性**: SMA25/75は遅行指標。クロス時点で値動きの大半が終了している
- **RSI < 70 は実質ノーフィルター**: 買いシグナル155〜167銘柄（70%超）= フィルターとして機能していない
- **エントリータイミングの集中**: 初日10銘柄一括 → 相関リスクが未管理

### 改善提案
- RSI条件を `50 < RSI < 65` に変更（上昇モメンタム確認 + 過熱前）
- SMAを12/26に短縮して反応速度を上げる
- 出来高条件追加（直近5日平均の1.2倍以上）
- 1日最大3銘柄、同一セクター2銘柄まで

## 2. リスクマネージャー

### 良い点
- 損切りルールが存在し、全トレードで遵守されている
- 1銘柄上限10%の設定でリスク分散の意図はある
- 全トレード記録の透明性

### 問題点
- **分散の錯覚**: 10銘柄一括エントリーは分散ではなく過剰投資。6銘柄同時ストップアウトが証明
- **数学的に破綻**: 勝率12.5%で損益分岐するには平均利益¥7,700（約+25%）が必要。+7%半分利確では絶対に届かない
- **即再エントリーの悪循環**: NTN3回エントリーは戦略の欠陥を隠蔽する悪習
- **RR比 < 1**: 損切り-5%に対し利確+7%半分 → リスクリワードが逆転

### 改善提案
- 同時保有を10→5銘柄に削減、週あたりエントリー上限2銘柄
- ストップアウト銘柄は2週間再エントリー禁止
- RR比 ≧ 2 を最低基準に
- 勝率30%以上 + 月次プラス3ヶ月連続まではペーパー継続

## 3. マーケットアナリスト

### 市場環境
- 3月4日に日経平均2,033円安（歴代5番目）、3営業日で7.8%下落
- 中東情勢悪化によるリスクオフ相場
- トレンドフォロー戦略にとって**最悪のタイミング**

### 戦略との相性
- 「225銘柄中155銘柄が買いシグナル」は明らかに異常。健全な戦略なら10〜20%に絞られるべき
- 実際のエントリーRSIは30〜37（売られすぎゾーン）→ 下落途中の銘柄を大量に拾っている
- 鉄鋼3銘柄同時エントリーでセクター集中リスク顕在化

### 切り分け結論
| 要因 | 寄与度 |
|------|--------|
| 市場環境 | **30%** — 急落局面でトレンドフォローには不向き |
| 戦略設計の欠陥 | **70%** — フィルター甘すぎ、セクター分散なし、ボラティリティフィルターなし |

安川電機の成功例から「改善すれば勝てる戦略」の潜在性はある。

## 4. クオンツトレーダー

### 統計分析
```
期待値 = (0.111 × ¥1,224) - (0.889 × ¥1,100) = -¥842/トレード
プロフィットファクター = ¥1,224 / ¥8,801 = 0.139
損益分岐に必要なRR比 = 8.0（現状 1.11）
二項検定 p値 ≈ 0.004 → 低勝率は偶然ではなくシステマティックな問題
```

### 改善提案（数値込み）
- ストップを ATR(14) × 2.0 に変更 → 勝率11%→25-30%見込み
- 利確を+7%半分→+10%全量 or +15%半分に引き上げ → RR改善
- ADX(14) > 25 でトレンド強度フィルター追加
- ケリー基準で現状は「賭けるべきでない」状態
